# Calculator
A simple and interactive calculator built using HTML, CSS, and JavaScript.
This project performs basic arithmetic operations such as Addition, Subtraction, Multiplication, and Division.

<h1><u>Features</u></h1>
1.Clean and responsive UI
2.Supports +, −, ×, ÷
4.Error handling (division by zero, invalid input)
5.Easy to understand code structure

<h1><u>Technologies Used</u></h1>
HTML5 – Structure
CSS3 – Styling
JavaScript (ES6) – Logic/Functionality

<h2>This project is open-source and free to use.</h2>
